# 🎨 Paint Together : Live at : https://paint-together-hchq.onrender.com/

A real-time collaborative drawing web application where multiple users can draw, chat, and interact on a shared canvas.

---

## 🚀 Features

- 🧑‍🤝‍🧑 **Real-time Collaboration** – Multiple users can draw simultaneously  
- 🏠 **Room-Based System** – Create or join rooms using unique codes  
- 🎨 **Drawing Tools** – Brush, Eraser, Color Picker, Adjustable Brush Size  
- 💬 **Live Chat** – Communicate with users in the same room  
- 🔗 **Shareable Room Link** – Invite others easily  
- 💾 **Save Canvas** – Download your artwork  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Real-time Communication:** Socket.io
